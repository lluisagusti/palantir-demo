# PALANTIR DEMO — Intelligence Géopolitique en Temps Réel

Un tableau de bord 3D interactif et haute fidélité visualisant les risques géopolitiques mondiaux, les événements environnementaux et les infrastructures stratégiques en temps réel.

## 🌍 Aperçu

PALANTIR DEMO offre une vision complète de l'intelligence mondiale en suivant :
- **Menaces Géopolitiques** : Surveillance en temps réel des zones de conflit mondiales et des niveaux de risque.
- **Surveillance Environnementale** : Suivi en direct des séismes, volcans, impacts de foudre et niveaux de radiation.
- **Logistique Mondiale** : Visualisation en temps réel des vols d'avions et des routes de navigation maritime.
- **Données Météorologiques** : Modèles météorologiques en direct et conditions atmosphériques.
- **Surveillance** : Caméras de streaming public intégrées depuis des emplacements stratégiques à travers le monde.

## 🚀 Technologies

- **Framework** : React 19 (Vite)
- **Visualisation 3D** : `react-globe.gl` (propulsé par Three.js)
- **Stylisation** : Tailwind CSS 4.0
- **Icônes** : Lucide-React
- **Moteur Temps Réel** : Moteur de simulation personnalisé haute performance.

## 🛠️ Prise en Main

### Prérequis
- [Node.js](https://nodejs.org/) (v20+ recommandé)
- [npm](https://www.npmjs.com/)

### Installation

1. **Cloner le dépôt :**
   ```bash
   git clone <repository-url>
   cd palantir-demo
   ```

2. **Installer les dépendances :**
   ```bash
   npm install
   ```

### Lancer l'Application

1. **Démarrer le serveur de développement :**
   ```bash
   npm run dev
   ```
   *Accédez au tableau de bord via l'URL fournie par Vite (généralement http://localhost:5173).*

## 📈 Fonctionnalités Clés

- **Globe 3D Dynamique** : Visualisation haute performance avec exploration interactive des couches de données mondiales.
- **Surveillance Avancée d'Événements** : Modales détaillées pour chaque événement (séismes, vols, menaces) avec informations contextuelles.
- **Opérations Maritimes** : Suivi en temps réel des navires de fret et des pétroliers via les routes maritimes internationales.
- **Surveillance en Direct** : Système interactif de sélection de caméras pour visionner des flux publics dans le monde entier.
- **Esthétique "War Room"** : Interface utilisateur sombre élégante et contrastée, conçue pour une surveillance professionnelle du renseignement.
