## 🚀 快速启动

<div align="center">

### 本地测试

```bash
# 克隆情报平台
git clone https://github.com/lluisagusti/palantir-demo.git

# 安装依赖
cd palantir-demo && npm install

# 启动开发服务器
npm run dev
```

**访问:** `http://localhost:5173`

## 📊 功能深度解析

### 🌍 地球仪可视化

**多层数据叠加:**
1. **威胁环** - 地缘政治事件的脉冲警报
2. **飞行路径** - 显示飞行器移动的动画弧线
3. **海事路线** - 主要航运路线及交通数据
4. **天气图标** - 19 个城市的实时状况
5. **摄像头标记** - 直播位置带脉冲效果
6. **辐射监测** - CPM 读数带警报指示器

**交互元素:**
- 点击任意标记查看详情
- 拖动旋转地球仪
- 滚动/捏合缩放
- 空闲时自动旋转

---

**60+ 地点包括:**

| 🦁 野生动物 | 鹰巢、熊、海洋生物 |
| 🏛️ 地标 | 自由女神像、悉尼歌剧院 |

**功能:**
- YouTube 集成
